# Raihan Fadil Akbar - Web Portfolio

This is a premium, modern web portfolio tailored for a Mechanical Engineering & Technical Drafter role. It utilizes HTML, CSS, and JS with a dark mode aesthetic.

## How to host this on GitHub for free (GitHub Pages)

Follow these exact steps to make your portfolio live on the internet!

### Step 1: Create a Repository
1. Go to [github.com](https://github.com) and log in.
2. Click the `+` icon in the top right corner and select **New repository**.
3. Name the repository exactly: `your-username.github.io` (replace "your-username" with your actual GitHub username, e.g., `rafaraihan.github.io`). 
   - *Note: If you want it to look like `yourusername.github.io/portfolio`, you can just name it `portfolio`.*
4. Make sure it is set to **Public**.
5. Click **Create repository**.

### Step 2: Upload Files
1. On the new repository page, click the link that says **"uploading an existing file"**.
2. Open your computer's file explorer and find this `portfolio` folder.
3. Drag and drop **all the contents** of the `portfolio` folder (not the folder itself, but the files inside: `index.html`, `style.css`, `script.js`) into the GitHub upload area.
4. Click **Commit changes** at the bottom.

### Step 3: Enable GitHub Pages
1. In your GitHub repository, click on the **Settings** tab at the top.
2. In the left sidebar, scroll down and click on **Pages**.
3. Under **Build and deployment** -> **Source**, make sure it says "Deploy from a branch".
4. Under **Branch**, select `main` (or `master`), and `/ (root)`. Click **Save**.

### Step 4: Go Live!
- Wait about 1-2 minutes.
- Your website will now be live at `https://your-username.github.io` (or `https://your-username.github.io/portfolio`)!
